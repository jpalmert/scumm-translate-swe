# Swedish Translation Glossary

## Translation Philosophy

**General rule**: Translate descriptive names. Keep only non-descriptive proper names in English.

**Tone and style**: 
- **Prioritize tone over accuracy** - if it works better in Swedish and maintains the feel, use it even if historically inaccurate
- **Flair and playfulness first** - Monkey Island is a comedy with sarcastic humor. Keep it fun, colorful, entertaining
- **Evocative over literal** - choose words Swedish gamers will immediately "get" and enjoy
- **Sarcasm and wit** - preserve Guybrush's sarcastic commentary and the game's self-aware humor

**Exceptions**: Names that appear in hardcoded graphics stay in English in the graphics, but translate in dialogue/text:
- LUCASFILM GAMES - keep as-is everywhere
- Mêlée Island - keep as-is everywhere
- SCUMM BAR - keep as-is everywhere
- STAN'S PREVIOUSLY OWNED VESSELS - graphic stays English, but dialogue uses **"Stans Begagnade Fartyg"**

### Ship Names - Translate (Descriptive)
- **Sea Monkey** → **Havsapan**
- **Ghost Ship** → **Spökskepp**

### Island Geography - Translate (Descriptive)
- **Lookout Point** → **Utkiksplatsen** (more natural for a location than "utkikspost" which implies a manned post)
- **Governor's Mansion** → **Guvernörens herrgård** (manor - better than "villa" for grandeur)

### Brand References / Trademark Symbols

The SCUMM engine uses `\153` to render ™. The initial translation incorrectly used `®` throughout — this was wrong both legally (these are fictional in-game brands, not registered trademarks) and technically (the SE font has no ® glyph for `\015`).

**Rule: Always use ™ (rendered via `\153` in SCUMM) for in-game brand references.**

- **Monkey Island™** — game title and island name
- **Mêlée Island™** — island name
- **LOOM™** — reference to another LucasArts game
- **Davey Jones™ Lockers** — in-game brand
- **GRIPMASTER™**, **BREATHMASTER™**, **DIGMASTER™**, **SLASHMASTER™** — in-game brands
- **Blimpkameran™** — in-game brand (not "Blimkameran")

## Common Recurring Terms

### Pirate/Nautical Terms
- pirate → **pirat** (international term)
- ship → **skepp**
- crew → **besättning** (or **manskap** for archaic flavor)
- captain → **kapten** (never "kapen" — that's dialect/slang that looks like a typo)
- sail → **segla**
- treasure → **skatt** (can use **klenod** or **juvel** for variety)
- sword → **svärd**
- island → **ö**
- sea → **hav/havet**
- dock → **kaj/hamn**
- cabin → **hytt**
- mutinous → **myterisk** (not "muterande" which means "mutating")

### Pirate Ranks/Types
- buccaneer → **sjörövare**
- swashbuckler → **äventyrare / svärdsvingare**
- scallywag → **slyngel / drummel** (playful pirate insult, not "skurk" which is too villainous)
- scurvy dog → **skörbjuggsträk / din skabbiga hund**
- landlubber → **landkrabba** (not "landlansen" which is a non-word)

### Pirate-Flavored Insult Suffixes
The English uses "buddy" variations like "pal" and creates pirate-flavored compounds. The initial translation invented Swedish compounds with `-ansen`/`-gransen` that don't exist and sound awkward:

| English | Initial (wrong) | Correct |
|---------|-----------------|---------|
| buddy/pal | kompansen | **kompis** |
| baldhead insult | blankskansen | **blankskallen** |
| cannonball-head | kanonkulleskansen | **kanonkulan** |
| funny names for Guybrush | Gransen Dansen, Snansen | **Gyckelbull Tredsvansen, Treepåpp** (playful mangling) |

**Rule:** Don't invent Swedish word compounds that don't exist. Use real Swedish slang or natural nicknames.

### Recurring Character Descriptions
- **Important-looking pirates** → **Förnäma pirater** (pirate leaders in Scumm Bar — emphasize appearance/pomposity, not just "viktiga"; "viktigt utseende pirater" is a calque)
- **Men of Low Moral Fiber** → **Män av Tveksam Vandel** (street corner pirates — natural Swedish expression for dubious character)

### Game-specific Terms

- trial/test → **prövning** (prefer for epic Three Trials) or **prov** (for casual tests)
- quest → **uppdrag**
- riddle → **gåta**
- clue → **ledtråd**
- "pieces of eight" → **dubloner/dublon** (historically inaccurate but evokes pirates perfectly in Swedish — tone over accuracy. NEVER "åttor" which means "eights" literally and has no pirate association)
- "Jolly Roger" → Jolly Roger (historical pirate flag name, keep in English)
- stool (bar seat) → **stop** (never "mugg" which means mug/cup)

### Object Names

- **Idol of Many Hands** → **Idolen med de Många Händerna** — "idol" is correct Swedish for a worshipped statue (SAOL lists this meaning); context in a pirate/cannibal game disambiguates from the celebrity meaning. "Avgud" (false god) carries theological judgment unsuitable for the comedy tone
- **rubber chicken with a pulley in the middle** → **gummikyckling med en trissa i mitten** — "trissa" is the common word for a small pulley; "talja" is technical sailing rigging
- **piranha poodle** → **pirayapudel** — the pun is piranha + poodle; "piranjapadel" loses the joke
- **breath mints** → **mintpastiller** (not "andningsmyntor" which is a literal calque)
- T-shirt → T-shirt (modern item, international word)

**Translate normally:**
- sword → svärd
- map → karta
- key → nyckel
- compass → kompass
- shovel → spade
- rope → rep
- bottle → flaska
- root → rot (voodoo root)
- grog → **grogg** (double g — Swedish spelling; never "grog")

### Food/Drink
- grog → **grogg** (Swedish spelling has double g)
- root beer → **rotöl** (Swedish compound: rot=root, öl=beer; more accessible than leaving "root beer" in English; used consistently throughout)
- stew → **gryta**
- meat → **kött**
- fish → **fisk**

### Voodoo/Magic Terms
- voodoo → **voodoo** (international term; compound words: **voodookärlekspärlor**, not "voodoo kärlekspärlor")
- spell → **besvärjelse/trollformel**
- magic → **magi**
- ghost → **spöke** (prefer over "ande" which is spirit/soul)
- curse → **förbannelse**

## Special Cases

### "SCUMM" References
The word "SCUMM" appears in:
- "Scumm Bar" - keep as is (it's the game engine name)
- Mentions of "scumm" as ingredient in grog - keep for consistency

### The game title
"the Secret of Monkey Island™" → **"Monkey Islands™ Hemlighet"** (capital H — it's a title)
"the secret of Monkey Island™" → **"Monkey Islands™ Hemlighet"** (still capital H when referencing the legendary secret)
Swedish word order: possessor first. Note the possessive 's' on Island.

### Humor Preservation
Certain names are jokes that work in English:
- **Men of Low Moral Fiber** - translate concept, not literal words → "Män av Tveksam Vandel"
- **Previously Owned Vessels** - euphemism for "used ships", translate the euphemism

### Control Codes (NEVER TRANSLATE)
- `\255\003` — pause/line break
- `\255\006\NNN\000` — variable substitution  
- `\153` — trademark ™ (used after all in-game brands: Monkey Island™, LOOM™, etc.)
- `\250` — non-breaking space

### Dialog Attribution Codes (DO NOT TRANSLATE)
- `(D8)` - Guybrush dialog
- `(14)` - NPC dialog
- `(94)` - Various
- `(FA)` - Player choice
- `(13)` - Name display
- Other hex codes

## Consistency Notes

### Titles and Honorifics
- Governor → **Guvernör** (never "guvernören" with article in names/titles)
- Captain → **Kapten** (never abbreviated "Kapen")
- Sword Master → **Svärdsmästaren** (note the s-joint: svärd**s**mästaren)
- Sheriff → **Sheriff**

### Time References
- hours → timmar
- days → dagar  
- minutes → minuter
- "Meanwhile..." → "Under tiden..."

### Measurements
Keep pirate-era measurements:
- pieces of eight (currency) → **dubloner** (see Game-specific Terms)
- feet (measurement) → **fot**

### Initials in Signed Notes
Character initials must match their Swedish name:
- Herman Toothrot signs as "H.T." in English → **"H.R."** (Herman Rötbett)
- Lawyer firm "Yammer, Hem and Haw" → **"Gnäll, Hum och Hav"**

## Style Guidelines

1. **Overall Tone**: Playful, colorful, fun! This is a comedy game — choose vivid words with character over safe/generic translations. "Drummel" beats "skurk", "dubloner" beats "åttaöring", etc. **Entertainment beats accuracy.**

2. **Pirate Speech**: Use slightly archaic/colorful Swedish. Think "djärvt", "förbaskat", "tusan" rather than modern neutral language. Pirates should sound like pirates, not office workers. Make them entertaining characters.

3. **Guybrush's Voice**: Modern, slightly sarcastic, often self-deprecating. He's earnest but also comments on the absurdity around him. **Preserve his sarcasm and wit** — these are key to the character.

4. **Formality Levels**:
   - Governor Marley: Formal/educated Swedish (but not stuffy)
   - Pirates: Casual, colorful, slang — they're criminals with personality
   - Herman Rötbett (Herman Toothrot): Rambling, slightly crazy, tangential
   - Stan: Fast-talking salesman — energetic, over-the-top, relentless. Uses "vi snackar" not "vi pratar", "kompis" not formal address, rapid-fire sales patter. Every sentence should feel like pressure.
   - Fester Shinetop: Menacing authority figure — short, threatening sentences

5. **Humor and Sarcasm**: 
   - Prioritize funny over literal
   - **Preserve sarcastic tone** — key to the game's humor
   - If a joke doesn't work, create a NEW Swedish joke
   - Better a good Swedish joke than a failed translation of an English one
   - Self-aware humor and fourth-wall breaks should stay funny in Swedish

6. **Sloppy speech (ja/mej/dej)**: Only use informal spellings when the English itself is grammatically incorrect or extremely casual. Standard dialog uses jag/mig/dig even for pirates. The MI1 exception: Stan's dock delivery scene in room 083 where he speaks fast and sloppy — ja/mej/dej was used there but was WRONG; all instances were corrected to jag/mig/dig. **Default: always jag/mig/dig.**

## Notes on Object Name Consistency
When an object appears as both OBNA (object name) and in dialog:
- **fabulous idol** / **Idol of Many Hands** — same object, use "idol" consistently (never "avgud")
- **rubber chicken with a pulley in the middle** — iconic phrase, always "trissa" (never "talja")
- Ship names in object vs. dialog must match (e.g. "Havsapan" everywhere)
- **stool** (bar seat) — always "stop" in both OBNA and dialog (never "mugg")
- When an OBNA entry and its VERB entries refer to the same object, the noun must be the same word
